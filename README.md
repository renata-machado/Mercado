# Sistema de Recomendação de Promoções

O sistema tem como objetivo automatizar a análise de vendas e estoque para sugerir promoções de forma rápida e prática. A partir de arquivos CSV enviados pelo usuário, ele identifica oportunidades de promoção com base em métricas e fluxo de vendas. Dessa forma, as promoções podem ser geradas automaticamente em cards prontos para divulgação, tornando o processo mais eficiente e ágil.



## Funcionalidades

- FastAPI - criação da API e endpoints do sistema.
- Streamlit -  interface web para visualização de métricas e promoções.



## Como funciona


- O usuário envia os arquivos CSV com dados de estoque e vendas.
- O sistema processa os dados, calcula métricas e identifica oportunidades de promoção.
- Cards promocionais são gerados.
